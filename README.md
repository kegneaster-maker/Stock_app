
# 📈 Stock Market Prediction System
A machine learning-based web application that predicts stock price trends using historical data. Built with Python and deployed via Streamlit.

# Overview
This project aims to provide financial insights by analyzing historical stock market data. Using time-series forecasting, the model predicts future prices, helping users make data-driven investment decisions.
# Features
Real-time Visualization: Interactive charts showing historical stock performance.

Trend Prediction: Forecasts future prices based on selected timeframes.

User-Friendly Interface: A clean dashboard built with Streamlit for non-technical users.

Technical Analysis: Includes indicators like Moving Averages to support predictions.
# 🧪 Tech Stack
**Language**: Python

**Data Analysis**: Pandas, NumPy

**Machine Learning**: Scikit-learn (Regression/Time-series models)

**Visualization**: Matplotlib, Plotly

**Deployment**: Streamlit Cloud
